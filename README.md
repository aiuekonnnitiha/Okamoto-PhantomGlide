# Okamoto-PhantomGlide
🚀 Okamoto Fly (Okamoto Fly Script)
"He who conquers the sky, conquers the game!" This script is the ultimate flight tool developed to give you absolute mobility by allowing your character to fly freely!  
🔥 Key Features
• Complete Flight Control: You can soar through the sky in any direction—up, down, left, or right!  
• Adjustable Speed: Change your flight speed instantly using the + and - buttons or the input box!  
• R6 / R15 Compatibility: Optimal flight logic is automatically applied regardless of your avatar type!  
• Sleek UI: Features a dark-themed UI with Gotham fonts, supporting draggable movement and minimization!  
🛠️ Controls
1. FLY Button: Toggles flight mode ON and OFF.  
2. UP / DOWN Buttons: Adjusts your altitude. Hold them down for continuous movement!  
3. Speed Settings: Enter a value in the text box or use the +/- buttons to tune your velocity.  
4. Minimize (-): Folds the UI to keep your screen clear.
